# borneohackwknd

Veri - EUDR Compliance Demo for Borneo Hackathon

## Get Started

**Prerequisites:** Install [Node.js](https://nodejs.org/), [Python](https://www.python.org/), and [Tesseract OCR](https://github.com/tesseract-ocr/tesseract)

**One-Click Setup:** Run `setup.bat` to install all dependencies automatically.

## Quick Start

**Backend:**
```bash
cd src/backend
python main.py
```
Backend runs on http://localhost:8002

**Frontend:**
```bash
npm install
npm run dev
```
Frontend runs on http://localhost:5173

## Structure
- `src/frontend/` - React + TypeScript UI
- `src/backend/` - FastAPI server + Firebase
